# branch 運用

## main 
- 常に動作する状態のコードのみをマージするブランチです。
- リリース可能な安定版コードを管理します。

## dev
- mainにマージする前のワンクッションとして使用する開発ブランチです。
- 新機能や修正を加えた際に、まずこのブランチで動作確認を行います。

## feature/aws/handson
- AWSのCloudFormationハンズオン実施時に作成した機能ブランチです。
- ディレクトリ構成の準備後にすぐ切られており、ハンズオン内容以外の変更は加えていません。
